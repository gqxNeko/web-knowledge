## 1.所有元素可继承属性

* 元素可见性
  * visibility
  * opacity
* 光标属性
  * cursor

## 2.不可继承属性

* display
* 文本属性
  * vertical-align
  * text-shadow
  * white-space
  * unicode-bidi
* 盒子模型属性
  * width
  * height
  * padding
  * margin
  * border
* 背景属性
  * background-image
  * background-color
  * background-position
* 定位属性
  * float
  * clear:both
  * position
* 生成内容属性(不熟悉)
  * content
  * counter-reset
  * counter-increment
* 轮廓样式属性
  * outline-style
  * outline-width
  * outline-color
  * outline
* 页面样式属性
  * size
  * page-break-before
  * page-break-after

> 补充：a标签的字体颜色不能被继承
> h1-h6标签小的大小不能被继承
